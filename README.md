# Sadao Customs Queue System - Online Ready

## วิธี Deploy ขึ้น Render.com ฟรี

1. อัพโปรเจคนี้ขึ้น GitHub
2. เข้า Render.com → New → Web Service → Connect GitHub repo
3. ตั้งค่า:
   - Build Command: `npm install`
   - Start Command: `npm start`
4. กด Deploy รอ 2-3 นาที

## ลิงก์ใช้งาน
- หน้าแรก: `https://your-app.onrender.com`
- จอแสดงคิว: `https://your-app.onrender.com/QueueDisplaySadao_v2.html`
- จอกดคิว: `https://your-app.onrender.com/SadaoCustomsQueue2569_v2.html`

## รันในเครื่อง
```
npm install
npm start
```
เปิด http://localhost:3000
